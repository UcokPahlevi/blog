Udah Gede Aja

Kalo kalian berpikir bahwa saya lahir langsung gede, kalian g*blk, orang jelas- jelas semua orang pasti pernah kecil, jangankan orang monyet aja kecil dulu baru gede.

Sengaja ya di foto ini gaya nya di sama in kayak waktu kecil, soalnya biar bisa ngebeda in aja gitu sama atau ngga saya yang sekarang sama saya yang kecil, menurut kalian gimana, sama atau beda? Jawab dalam hati aja. Kalo gapunya hati gimana? ya mati lah.

Difoto itu antara saya umur 14 atau 15 saya lupa. Yang saya inget disitu saya udah lulus SMP dan otw SMA tapi saya ambil SMK, dan sedikit info bahwa disitu saya sudah menjadi manusia yang benar - benar ateis. Menurut saya, saya yang disitu lebih ganteng dari pada saya yang waktu kecil, menurut kalian gimana? Gausah dijawab juga gapapa kalo masih gapunya hati.

Kalo kalian gapernah liat saya waktu kecil sekarang saya kasih liat deh.

Saya cuma punya foto ini, gapunya foto lagi. Padahal saya dulu sering difoto sama tetangga saya waktu saya kecil, tapi tetangga saya udah meninggal jadi bingung mau minta fotonya gimana. Tapi kalo saya niat lebih buat dapetin foto saya waktu kecil kayaknya saya bisa cari ke sekolah SD saya cuma niat lebih nya gaada. Urusan saya waktu kecil itu ga penting, kita ngoongin yang udah gede aja, udah gede ngapai aja?

Seperti orang - orang gede pada umumnya, ga semua sih. Tapi ya saya selama ini ga ngapa - ngapain cuma nikmatin masa muda aja, soalnya saya punya keyakinan kalo saya gabakal tua. Maka dari itu saya nikmatin masa muda saya semaksimal mungkin, gimana caranya?

Cara orang nikmatin masa muda pasti beda - beda ya, kalo saya nikmati dengan nonton tiktok dikamar sambil sesekali main game, gitu aja tiap hari, kasian kan? Jelas. Tapi untungnya saya disitu bohong.

Ngapain Aja?

Sekolah

Karena saya masih sekolah ya mau gamau saya harus sekolah. Ngapain aja disekolah? Nyari doraemon.

Main

Masa iya, masa muda ga dipake main. Kebanyakan di masa muda saya ya main, biar apa? Biarin.

Main Lagi

Bedanya main sama main lagi apa? gaada bedanya cuma biar judul nya ga sama aja.

Main Terus

Main aja terus, emang di marahin mama? oh saya gapunya mama, tapi ibu punya.

Main - main

Kalo dilanjut masih banyak, intinya semua di masa muda saya, saya pake buat main.
